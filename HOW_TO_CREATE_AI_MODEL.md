## â— Troubleshooting

### "Rate limit exceeded" Error
- **Solution**: Wait 2-3 minutes, then try again
- OpenAI has rate limits on API calls
- Your key is valid, just temporarily throttled

### "Model not found" Error
- **Solution**: Double-check the Model ID is exactly `gpt-4o-mini`
- No spaces, all lowercase
- Must use hyphen (-) not underscore (_)

### "Invalid API Key" Error
- **Solution**: Copy the entire API key again
- Make sure no spaces before/after
- Key starts with `sk-proj-`
